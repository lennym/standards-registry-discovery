One benefit of using information standards is that it allows technologies owned by different care settings to see and use one another’s data in real-time. This is known as interoperability.

Interoperability can transform whole industries. We can move money quickly between banks because the banking industry uses universal standards to exchange financial information.

In a health context it can allow, for example, hospital staff to see and add to someone’s GP records instantly instead of relying on staff re-entering data into systems.

Interoperability is not the same as manually integrating systems that have been built to different specifications, which can be time-consuming, expensive and doesn’t work on a national scale.

Interoperability standards

It will take time to achieve interoperability which is why some parts of the health sector are focussed on achieving specific data-sharing goals. For example, programmes such as Transfer of Care have been created to help standardise and share discharge information for people leaving hospitals.

The term interoperability standards is sometimes used to refer to all the different standards that need to be used together to achieve these goals.
