Get involved with community discussions

The standards community platform is designed to help bring people together to solve issues around standards development, implementation and adoption.

It is open to anyone who uses, develops, commissions, owns or assures standards, services and APIs relating to interoperable technology in health and social care.

Discussions forum on GitHub

In the discussions forum, you can:

  • read and comment on discussions on use cases, standards implementation and local specifications
  • post a question or share your own experience in a new discussion
Go to the discussions forum on GitHub

How the community platform works

The platform is hosted on GitHub, an open source platform designed for collaboration.

You will need a public GitHub account if you want to comment, start a discussion or make a contribution to the standards community development backlog.

You do not need to create a GitHub account to browse and search.

Support

The standards directory team moderate the standards community development backlog and the discussions forum.
