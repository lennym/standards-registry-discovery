Standards directory

Discussion forum

The discussion forum is open to anyone with an interest in standards adoption and interoperability in health and social care.

You can:

  • read and contribute to discussions, for example around implementing standards or developing local specifications
  • ask the community a question
Go to the discussion forum on GitHub

(Opens a new window)

How the discussion forum works

The platform is hosted on GitHub, an open source platform designed for collaboration.

You will need a public GitHub account if you want to comment or start a discussion.

You do not need to create a GitHub account to browse.

Support

The standards directory team moderate the discussions forum. You can contact them by emailing standards.directory@nhsx.nhs.uk.
